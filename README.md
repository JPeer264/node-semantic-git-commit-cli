## Usage

```sh
$ npm i -g MODULE_NAME
```
or
```sh
$ yarn add global MODULE_NAME
```

After installation it is available in your shell as:
```sh
$ semantic-git-commits
```
or even shorter:
```sh
$ sgc
```
